About this project:-

This is a medicine finder website which helps you to find alternative medicines for your prescribed medicines.

This website is made using HTML, CSS and JavaScript. The data of the medicines is taken from various sources and compiled into a single database.

More features will be added in the future. If you have any suggestions or feedback, please contact us.

Thank you for visiting our website!
